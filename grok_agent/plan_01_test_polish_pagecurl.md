# Plan 01: Test & Polish PageCurl (2-Pass)

**Phase:** B2 (Multi-pass)  
**Priority:** High  
**Est. Sessions:** 1–2

## Goal
Make the new 2-pass PageCurl look excellent and feel like classic Milkdrop.

## Tasks
1. Test across different aspect ratios, speeds, and preset combinations
2. Tune highlight intensity, glow radius, and drop shadow strength
3. Add subtle noise or edge softening if needed for more organic feel
4. Verify performance impact (especially on Emscripten)
5. Create before/after comparison (optional but recommended)

## Success Criteria
- PageCurl looks noticeably better than the old single-pass version
- No visual artifacts at extreme aspect ratios
- Runs smoothly on both desktop and web

## First Step
Run the current PageCurl at various speeds and aspect ratios. Note what looks good and what needs tuning.
